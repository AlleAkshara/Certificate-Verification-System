# Blockchain Certificate Manager

A decentralized application for secure certificate storage and verification using blockchain technology.  

---

##  Project Overview
This project leverages blockchain to solve the challenges of traditional certificate verification.  
By storing certificate data on an immutable ledger, the system ensures:
- Authenticity and tamper-resistance
- Instant verification without intermediaries
- Reduced administrative overhead
- Transparency and global accessibility

---

##  Key Features
- **Upload Certificates**: Store certificate metadata and file hashes securely on the blockchain.  
- **Verify Certificates**: Instantly check authenticity by comparing uploaded files with blockchain records.  
- **View Certificates**: Browse all certificates stored in the system for transparency.  
- **Privacy Control**: Users can selectively share credentials while maintaining ownership of their data.  

---

##  Challenges Addressed
- Forgery and tampering risks  
- Manual, time-consuming verification processes  
- High administrative costs  
- Lost or damaged certificates requiring reissuance 

---

##  Technology Stack
- Frontend: HTML, CSS
- Backend: Python, Flask
- Blockchain: Custom Blockchain Implementation
- Storage: JSON Ledger, Local File Storage
- Cryptography: SHA-256
